
# Todo Demo

Hi,
This app is a basically a to-do app that uses ngrok for json db.

To start application (after -npm install-):
- 
    1. npm run tunnel
    2. npm run db
    3. expo start

Config:
- The url from the tunnel should be replaced with the url from todos and users.
- todos -> /src/api/todos
- users -> /src/api/users
